# TextTyper
Программа для проверки скорости печати

Для использования нужно собрать Makefile:
```bash
cmake .
```
После этого необходимо собрать проект:
```bash
make
```
Далее нужно запустить сам проект:
```bash
./main text.txt
```

Для запуска также можно использовать bash-команду:
```bash
bash start.sh
```
